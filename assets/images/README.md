# Pasta para imagens do sistema

Adicione aqui as imagens utilizadas no projeto:

- Logo do sistema
- Imagens de fundo
- Fotos de perfil padrão
- Ícones personalizados
- Imagens de conteúdo

## Formatos recomendados:
- PNG para imagens com transparência
- JPG para fotografias
- SVG para ícones e logos
- WebP para otimização (quando suportado)
